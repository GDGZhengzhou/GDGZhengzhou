郑州GDG I/O Redux 2016

一个月前的 Google I/O 2016 是否仍然历历在目，虽然我们不能去 IO 大会现场但是郑州GDG社区这边举办了第一次 I/O Extended 直播之夜活动。按照GDG社区以往的惯例，会再举办一次 I/O Redux 的 I/O 回顾活动。如果您之前参加了郑州 GDG I/O Extended 直播之夜活动，那么这次不用熬夜就可以再次体会到 I/O 三天的技术与激情。

活动详情

  • 活动时间：2016年6月26日 下午 13:30 - 17:00
  • 活动地点：郑州市金水区鑫苑路路河南信息工程学校教学楼107室
  • 活动费用：免费
  • 活动安排：
    1. 签到
    2. Google I/O 2016 主要内容回顾
    3. 下一代计算平台——VR生产力
    4. 合影

关于 GDG

GDG , 即 Google Developer Groups 的缩写或谷歌开发者社区，是谷歌开发者部门发起的全球性技术社区。 GDG 是面向对 Google 和开源、开放技术感兴趣的公益性开发者社区，内容涵盖 Web/Chrome、Android 和其它 Google API 等。全国已成立了二十多个 GDG 社区遍布全国，赶快加入你身边的组织吧。
